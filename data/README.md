## Dataset Information :

The dataset used in this project is publicly available on Kaggle and is not included in this repository due to its large size.

### Dataset Source :
- **Name:** Toxic Comment Classification Challenge
- **Platform:** Kaggle
- **Direct link:** https://www.kaggle.com/datasets/julian3833/jigsaw-toxic-comment-classification-challenge/data

### How to Download :
1. Create a Kaggle account (if you don’t have one)
2. Go to the dataset page on Kaggle
3. Download the dataset files
4. Place the CSV file(s) inside this `data/` folder before running the notebook

Example :
```text
data/
├── train.csv
├── test.csv
